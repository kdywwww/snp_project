# 1. Artifact Registry 등록 절차

## 1.1. 로컬 도커 인증
gcloud auth configure-docker us-central1-docker.pkg.dev

## 1.2. 이미지 태그 지정
# 경로 형식: [리전]-docker.pkg.dev/[프로젝트 ID]/[저장소 이름]/[이미지 이름]:[태그]
docker tag snp-full-image us-central1-docker.pkg.dev/long-centaur-402106/snp-repo/snp-full-image:latest

## 1.3. 이미지 푸쉬
docker push [YOUR_REGION]-docker.pkg.dev/[YOUR_PROJECT_ID]/mlops-repo/vm-manager:latest
